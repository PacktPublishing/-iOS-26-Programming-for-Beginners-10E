<i>iOS 26 Programming for Beginners</i> returns in its 10th edition, fully updated to reflect Apple’s latest innovations in app development. Perfect for complete newcomers as well as those transitioning from other platforms, this practical guide walks you through every step of building your first iOS app using Swift 6 and Xcode 26.</br>
Following a project-based approach, you’ll create a feature-rich journal app while mastering UIKit fundamentals, including storyboards, navigation, data persistence, media integration, and map-based features. Each chapter builds on the last with clear explanations, step-by-step instructions, and real-world examples that reinforce your learning.</br>
Along the way, you’ll enhance your app with Apple Intelligence, Apple’s on-device AI platform, to deliver smart, personalized user experiences. You’ll also leverage the elegant Liquid Glass UI design system introduced in iOS 26, to give your app a modern and immersive feel.</br>
By the end of this book, you’ll have built and deployed a complete iOS app and gained the essential skills to pursue a career in mobile development.</details>

<details open> 
  <summary><h2>Requirements for this book</summary>
This book has been completely revised for iOS 16, macOS Tahoe 26, Xcode 26, and Swift 6. <i>Part 4</i> of this book also covers the latest technologies introduced by Apple during WWDC 2025, which are the updated design using Liquid Glass, Icon Composer, Apple Intelligence, and the Foundation Models framework.
To complete all the exercises in this book, you will need the following:
<ul><li>A Mac computer running macOS 15.0 Sequioa, macOS 26 Tahoe, or later</li>
    <li>Xcode 26.0 or later</li>
</ul>
To check whether your Mac supports macOS Tahoe 26, use this link: <a href="https://www.apple.com/my/os/macos/">https://www.apple.com/my/os/macos/</a>. If your Mac is supported, you can update macOS using <b>Software Update in System
Settings</b>.
To get the latest version of Xcode, you can download it from the Apple App Store. Most of the exercises can be completed without an Apple Developer account and use the iOS Simulator. If
you wish to test the app you are developing on an actual iOS device, you will need a free or paid Apple Developer account.
The following chapter requires a paid Apple Developer account: <i>Chapter 27</i>, <i>Test and Submit Your App to the App Store</i>. Instructions on how to get a paid Apple Developer account are included.
    
</details>
